## 分析演进与架构

### Stage 1: 综合评分法 (Heuristic Rating)
适用于赛事初期队伍实力悬殊大、互相交手记录较少的情况。
基于市场赔率、HLTV排名、近期战绩等打出 0-100 的综合分，通过 Bradley-Terry 拟合基准胜率后进行蒙特卡洛模拟。

### Stage 2: 双轨制矩阵融合 (Dual-Matrix Blend)
进入竞争白热化的 16 强淘汰赛阶段后，由于顶尖队伍存在极其严重的战术相克（“剪刀石头布”效应），传统的综合评分法会导致比分信息的严重流失。因此重构了底层架构：
1. **市场轨道**：提取 Polymarket 盘口构建包含市场共识的先验对决矩阵。
2. **数据轨道**：提取近半年所有历史比赛的具体比分，利用内置的“动态外部 Elo 系统”过滤刷分水分，构建时间衰减的对位矩阵。
3. **混合与校准**：7:3 混合市场与数据矩阵，定点覆写异常的首轮盘口，并利用 Bo1 和 Bo3 的历史胜率落差生成“战术深度修正因子”。
4. **高保真模拟**：严格还原 CS2 官方布赫霍尔茨 (Buchholz) 分池规则进行 20 万次蒙特卡洛模拟，并暴力搜索全局最优 Pick'Em 组合。

## 关键策略原则
- **强队应放3-0位**: 3-1/3-2位不含3-0结果，强队3-0概率高会"浪费"3-1/3-2位。
- **关联性风险**: R1对阵决定了pick之间的关联。选0-3队时优先避开与自己3-0 pick首轮对阵的队(爆冷会同时废两个pick)。
- **摒弃纸面战力**: 模拟器能在 Swiss 规则中发现“败者组红利”和“Bo1 幻神的 Bo3 噩梦”，找到被市场低估的战术强队。
